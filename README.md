# Dashboard для українців

Інформаційний дашборд з актуальною інформацією про повітряні тривоги, графік відключень електроенергії та прогноз погоди для України.

## Основне

Дашборд надає користувачам важливу інформацію в реальному часі:
- **Повітряні тривоги** - інтерактивна карта України з відображенням активних тривог по всіх областях та детальний список регіонів
- **Графік відключень** - актуальний графік відключень електроенергії для Київської області (група 5.2)
- **Прогноз погоди** - поточна погода та прогноз на 4 дні для Софіївської Борщагівки
- **Push-сповіщення** - можливість увімкнути сповіщення про тривоги та відключення

## Технології

Проєкт побудований на **Next.js** з використанням:
- React
- TypeScript
- Tailwind CSS
- Service Workers для PWA функціональності
- API Ukraine Alarm для даних про тривоги
- Open-Meteo API для даних про погоду

## Розробка

Проєкт періодично вдосконалюється та оновлюється новими функціями.

### Встановлення залежностей

```bash
pnpm install
```

### Запуск локального сервера розробки

```bash
pnpm dev
```

Відкрийте `http://localhost:3000` у браузері.

### Збірка для продакшену

```bash
pnpm build
pnpm start
```

## Deployment

Проєкт можна деплоїти на будь-який хостинг, сумісний із Next.js (рекомендується Vercel).

### Змінні середовища

Для роботи з API тривог необхідно налаштувати змінну середовища:

```
UKRAINE_ALARM_API_KEY=your_api_key_here
```
